PROBLEM DESCRIPTION:
Managing hospital operations manually is error-prone and time-consuming. This system simulates key hospital workflows using Object-Oriented Programming principles in Java. It handles patient registration, doctor consultations, appointment scheduling, prescription generation, billing, and administrative system reporting all through a clean, console-based interface.




HOW TO RUN THE CODE:

1.Download an IDE(like VScode or Eclipse).

2.Clone or download the repository.

3.Open the project and ensure all .java files are under the Case_Study package.

4.Run Main.java as a Java Application.
